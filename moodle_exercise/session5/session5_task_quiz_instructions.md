# Session 5 Task-by-Task Self-Check Quiz

This quiz is a practice and validation quiz for Session 5. Its purpose is to help you build your program step by step and check each task before you attempt the final full-program submission.

Session 5 focuses on positional arguments, keyword arguments, and default values. Each task quiz checks one task only. You should use these quizzes while developing your code in `session5/session5_iris_template.py`.

## Recommended workflow

1. Complete one task in your local Session 5 file.
2. Copy your current code into the matching `ses5_task` quiz in Moodle.
3. Use Moodle validation to check your code for that task.
4. Read the feedback and fix your code if needed.
5. Move to the next task only after the current task is correct.

## What each task quiz checks

- Task 1 checks `calculate_accuracy(correct=0, total=0)`.
- Task 2 checks `run_classifier_pipeline(threshold=2.0, print_each=True)`.
- Task 3 checks `run_prediction_loop(dataset, threshold=2.0, print_each=True)`.
- Task 4 checks the mixed positional-and-keyword call to `update_result_counts(...)`.
- Task 5 checks the default run block in `main()`.
- Task 6 checks the positional override run in `main()`.
- Task 7 checks the keyword override run in `main()`.
- Task 8 checks the reordered keyword run in `main()`.

## Important notes

- These task quizzes are for self-checking while you develop your code.
- Each task quiz checks only one small part of Session 5.
- Passing a task quiz does not automatically mean your whole Session 5 program is complete.
- Keep your work in one Python file as you progress from Task 1 to Task 8.

After all task quizzes are correct, move to the final complete-code submission quiz.
